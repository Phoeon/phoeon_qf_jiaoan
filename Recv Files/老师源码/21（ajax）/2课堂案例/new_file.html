//第一步 ： 解决Ajax的兼容问题 var xhr = window.XMLHttpRequest?( new XMLHttpRequest() ) : ( new ActiveXObject('Microsoft XMLHTTP') ) ; xhr.open('get','text.txt',true); //准备发送： 1.发送方法:get post form 1.action: 地址 2.method:get post 3. 2.url 3.是否异步： 同步(false) - 阻塞 异步(true) - 非阻塞 xhr.send(); //正式发送 xhr.onreadystatechange = function(){ if( xhr.readyState == 4 ){ alert( xhr.responseText ); } } xhr.onreadystatechange:readyState更改的时候触发 xhr.readyState:(ajax程序的处理过程); 0：请求未初始化（还没有调用 open()）。 1：请求已经建立，但是还没有发送（还没有调用 send()）。 2：请求已发送，正在处理中（通常现在可以从响应中获取内容头）。 3：请求在处理中；通常响应中已有部分数据可用了，但是服务器还没有完成响应的生成。 4：响应已完成；您可以获取并使用服务器的响应了。 xhr.responseText:返回的数据内容； xhr.status : 服务器状态码；
